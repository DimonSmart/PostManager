### Задание №3. Кнопка тестирования таргета (Telegram)

**Цель:**
Добавить в окно настройки таргетов кнопку тестирования, которая отправляет тестовое сообщение в выбранный таргет. На текущем этапе поддерживается только тип канала **Telegram**.

**Требования к UI:**

1. В списке таргетов (рядом с кнопками **Edit** и **Delete**) добавить новую кнопку: **Test**.
2. Кнопка отображается и активна только для таргетов типа **Telegram** (для остальных типов пока не требуется).

**Логика работы:**

1. При нажатии **Test** приложение отправляет **тестовое сообщение** в выбранный Telegram-таргет (в соответствующий канал/чат, указанный в конфигурации таргета).
2. После выполнения операции пользователю показывается результат:

   * **Успех:** сообщение вида “Тестовое сообщение отправлено успешно”.
   * **Ошибка:** сообщение вида “Ошибка отправки: <текст ошибки>”, где `<текст ошибки>` содержит диагностически полезную информацию (текст исключения/ответа API), чтобы было понятно, что пошло не так.

**Примечания:**

* Формат и текст тестового сообщения могут быть простыми (например: “Test message” / “Тестовое сообщение”), главное, чтобы оно было доставляемым и однозначно распознавалось как тест.
